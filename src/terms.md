## 2. Description of Service

### 2.1 Service Overview

DownBad is an uptime monitoring service designed for indie developers, small teams, and solo founders. The Service monitors your websites, APIs, and other online resources and notifies you when issues are detected.

### 2.2 Monitoring Types

The Service provides the following monitoring capabilities:

- **HTTP/HTTPS Monitoring** — Check website and API availability, response codes, and content
- **Ping Monitoring** — ICMP connectivity checks
- **Port Monitoring** — TCP port availability monitoring
- **DNS Monitoring** — Domain name resolution verification
- **SSL Certificate Monitoring** — Certificate validity and expiration tracking
- **WHOIS Monitoring** — Domain registration and expiration monitoring
- **Blacklist Monitoring** — Domain and IP reputation checking
- **Email Security Monitoring** — SPF, DKIM, and DMARC verification

### 2.3 Additional Features

Depending on your subscription plan, the Service may include:

- Multi-region monitoring
- Notification channels (email, webhooks, in-app)
- Public status pages
- Data export and archival
- Team collaboration features
